3a
Q3 (a) · 3 分 · 识别题
字数：50-80 字 · 段数：1 段 · 时间：~5 分钟

1结构公式

(a) 模板 ≈ 1-2 句直接回答 + 2+ 个直接引文短语（带「」）+ 价值/文化收束句（可选，加分）

2步骤分解

步骤 1 · 锁定题干关键词
"作者用了哪些数据" / "采取了哪些措施" / "哪些独特价值观"

题干的关键词决定答案的列举对象。识别题不需要推论，只需精准抓取选文中的明显信号。

步骤 2 · 找 2-3 处直接证据（带引号）
verbatim 引用，不要 paraphrase

从选文中找出 2-3 个最贴切的短引文。例如「1992年三百零三万人参加、录取率25%」「2006年九百五十万人报名、录取率56.8%」。引文必须带「」。

步骤 3 · 用 1-2 句话串联
语言简洁直接，避免冗长解释

把引文嵌入一两个连贯的句子。结构如「作者援引 A、B 两组数据，并以 C 为时间框定，奠定对比研究的基石」。

3夹缝实例 — 第 1 题 (a)

Sample · ~95 字

作者援引「1992年三百零三万人参加、录取率25%」与「2006年九百五十万人报名、录取率56.8%」两组高考数据，并以「跨越了二十三年的光景」框定研究跨度，加之「两所学校都属于二本高校」「都排在前20%」的可比性陈述，奠定了对比研究的统计基石。

4评分要素 checklist

  • ✅ 是否包含 2+ 直接引文（带「」）
  • ✅ 是否直接回答题干（不绕弯）
  • ✅ 是否在 50-80 字范围内（不超 100 字）
  • ✅ 是否避免泛泛概括（如「很多数据」「各种措施」）
  • 🔥 加分：是否在末尾加价值/文化收束句（如「奠定对比研究的统计基石」）
3b
Q3 (b) · 5 分 · 分析题（核心 TEEL 训练）
字数：200-250 字 · 段数：1 段（含 3 TEE 循环）· 时间：~12 分钟

1结构公式

(b) 模板 = [首先，技巧1 + 引文 + 效果 + 回扣题干] + [其次，技巧2 + 引文 + 效果 + 回扣题干] + [最后，技巧3 + 引文 + 效果 + 回扣题干]

关键约束：3 个 TEE 循环 / 3+ 命名技巧 / 每 cycle 必回扣题干关键词 / 每 cycle 至少 1 个直接引文

2每个 TEE 循环的内部结构

T (Technique)
命名一个具体技巧

非虚构文本必须用：对比论证 / 典型案例 / 数据论证 / 直接引语 / 反思性语言 / 细节描写 / 时间线编排 / 隐喻。禁用电影词汇（镜头 / 场景调度 / 蒙太奇 / 剧本 / 台词 / 对白）。

E (Example)
引文 + 上下文

verbatim 引用（带「」）+ 简短的上下文说明（"作者在描述海燕家庭时说......"）。引文是论据，不是装饰。

E (Explain)
解释引文产生什么效果

这个技巧 / 这个引文具体做了什么？避免「两张皮」——技巧名 + 引文不能孤立罗列，必须说明二者结合产生什么论证效果。

L (Link)
回扣题干关键词

每个 cycle 必须在结尾回扣题干（如题干说「理性分析态度」，则每 cycle 必有「理性」二字）。这是 5 分答案与 4 分答案的核心区分。

3夹缝实例 — 第 1 题 (b) 完整范文

Sample · ~225 字

首先，作者运用数据论证法，「1992年三百零三万、录取率25%」与「2006年九百五十万、录取率56.8%」的精确数字并置，将主观经验置于可验证的统计框架，体现了作者理性的研究底色。 其次，作者贯穿使用反思性语言，「我的直觉是」与「我以前总认为......但进入具体的分析和审视，可以发现」的自陈结构使分析在自我检视中推进，呈现边思考边求证的理性姿态。 最后，作者采取限定性表述，强调「两所大学都属于二本高校」「都属汉语言文学专业」，以同质化前提剔除生源差异，使比较得以聚焦于「学历含金量如何发生变化」这一理性命题。

4评分要素 checklist

  • ✅ 是否有 3 个独立 TEE 循环（首先 / 其次 / 最后）
  • ✅ 是否命名 3+ 具体技巧（非泛称如「修辞」「写法」）
  • ✅ 每个 cycle 是否回扣题干关键词（题干含「理性」→ 每 cycle 出现「理性」）
  • ✅ 每个 cycle 是否含 1+ 直接引文（带「」）
  • ✅ 是否在 200-250 字范围（超 300 字会被认为冗余）
  • ❌ 是否出现「两张皮」（技巧名 + 引文孤立罗列，无解释）
  • ❌ 是否使用电影词汇（镜头 / 场景调度 / 剧本 / 台词）
  • ❌ 是否使用 hedge 词（可能 / 也许 / 或许）
3c
Q3 (c) · 7 分 · 综合题（满分门）
字数：350-400 字 · 段数：3 mini-论点段 · 时间：~20 分钟

1结构公式

(c) 模板 = [首先，论点1：选文引文 + wider-text 引文 + 收束句] + [其次，论点2：选文引文 + wider-text 引文 + 收束句] + [最后，论点3：选文引文 + wider-text 引文 + 收束句]

硬约束（2025 NESA 真题验证）：不引 wider text = Band 4 封顶 automatically。无商量余地。

2每个 mini-论点段的内部结构

① 选文引文（必）
来自题目给定的 Text 1 节选

每个 mini 段必须从选文中引一句话作为锚点。这是「立足选文」的证据。

② Wider-text 引文（必 · 满分门）
来自题干指定的其他章节（如「结合第 3 章 + 第 6 章」）

必须 verbatim 引用其他章节的内容（带章节号 + 人物名 + 「引文」）。泛泛说「结合全书其他章节」不算——必须具体到「如海燕『从一名劳务派遣的漂泊女孩，变成了一名金融系统的正式员工』的身份转换」。

③ 收束句（差异化收束）
"X 与 Y 差异" / "体现 Z 转变" / "揭示 W 机制"

每个 mini 段以一个差异化命题收束——把引文之间的关系解释清楚。例如「教育价值并未消失而是『变形』——它不再保证起点，但仍能撬动转折」。

3夹缝实例 — 第 1 题 (c) 第三个 mini 段

Sample · ~140 字

最后，作者通过反思性立场揭示价值演变的多维度。选文中「我以前总认为......但进入具体的分析和审视，可以发现」的自我修正结构，承认了「文凭贬值的呼声几乎成为共识」却不被其裹挟。结合吴志勇「在他所从事的职业门类中，压根就没有稳定一说」的辗转与王国伟父亲「在四会买房」获得的家庭支撑两种轨迹可见，「时代夹缝中的光芒」揭示的不是教育的失效，而是其价值已从制度性兑现转向个体性激活的根本差异。

4评分要素 checklist

  • ✅ 是否有 3 个独立 mini-论点段（首先 / 其次 / 最后）
  • ✅ 每个 mini 段是否含 选文引文 + wider-text 引文
  • ✅ Wider-text 引文是否具体到章节 + 人物 + verbatim 引文（不能泛泛说「其他章节」）
  • ✅ 每个 mini 段是否有差异化收束句（"X 与 Y 差异" / "从 X 转向 Y"）
  • ✅ 总字数是否在 350-400 字范围（超 600 字会被认为啰嗦）
  • 🔥 满分门：所有 wider-text 引文必须 verbatim（不是 paraphrase）
  • ❌ 是否使用 hedge 词（可能 / 也许 / 或许）
  • ❌ 是否陷入剧情/事件复述（必须是分析而非概述）
4
Q4 · 25 分 · Cross-Text Thematic Essay
字数：~800 字 · 段数：4 段 · 时间：~50 分钟

1结构公式

Q4 模板 = 4-段 cross-text:
• Intro 50-80 字 — 背景 + 立场 + 文本介绍
• Body 1 ~350 字 — Text 1 单文本论证（3 个 mini TEEL 循环：首先 / 其次 / 最后）
• Body 2 ~350 字 — Text 2 单文本论证（同样 3 个 mini TEEL 循环）
• Conclusion 50-80 字 — 「综上所述」+ 综合升华

硬约束：一段一文本 / 每 Body 必含 3 TEEL 循环 / 综上所述必须出现 / 文体匹配（非虚构 = 非虚构词汇）

2Body 段内部 3 TEEL 循环结构

TEEL Cycle 1 · 首先
T → E → E → L

技巧1（如典型案例选择） + 引文 + 效果解释 + 回扣 essay 主题

TEEL Cycle 2 · 其次
不同维度的技巧

技巧2（如数据论证 / 反思性语言） + 引文 + 效果 + 回扣

TEEL Cycle 3 · 最后
深化到主题层

技巧3 + 引文 + 效果 + 回扣 + 段尾升华到 essay 中心命题

3跨文本 essay 注意事项

文体语体匹配（mandatory）：
  • 分析非虚构文本（夹缝中的光芒）→ 用「叙事手法 / 个案研究 / 数据论证 / 引文调度 / 反思性语言」
  • 分析电影（别告诉她 / 流浪地球2）→ 用「镜头 / 场景调度 / 构图 / 剪辑 / 蒙太奇」
  • 分析小说（二马 / 大江大河）→ 用「人物塑造 / 情节安排 / 叙事视角 / 对话设计」
  • 分析歌曲（漂向北方）→ 用「歌词意象 / 反复音节 / 呼告 / 隐喻」
禁止跨语体混用——分析夹缝不能用「镜头」；分析别告诉她不能用「剧本 / 台词」。

4夹缝 essay 示例 — Q4-1 完整范文

Intro · ~70 字

城市化的浪潮把中国年轻人推上了流动的轨道，而个人的精神世界往往在这种流动中被反复打磨与重塑。纪实文学《夹缝中的光芒》借大量个案揭示出迁徙的物质代价与情感重量，流行歌曲《漂向北方》则以歌词意象浓缩漂泊者的孤独与希冀，二者共同呈现迁徙对一代人的深层冲击。

Body 1 (Text 1 · 夹缝中的光芒) · ~270 字

首先，《夹缝中的光芒》通过典型案例的并置呈现迁徙的物质代价......（潘海燕一家「父亲到广州开车」「母亲到佛山周记茶餐厅当服务员」的双线务工切片 + 直接引语「不论接到妈妈的第一封信......都因为思念而泣不成声」）其次，作者运用数据论证细化迁徙的结构性影响......（「五十二名同学，全部在广东就业」「定居广州的十七名」）最后，作者通过反思性结构升华迁徙的精神维度......（「中国乡村城镇化的进程......在不知不觉中自动完成」）

Body 2 (Text 2 · 漂向北方) · ~270 字

其次，《漂向北方》以歌词的浓缩意象呈现漂泊者的孤独感......（「漂向北方，别问我家乡」的反复 + 「妈我好想你」呼告 + 「黄沙像耳光」「梦想和现实的天秤」比喻）

Conclusion · ~75 字

综上所述，两文本以不同的语体——一边是数据与个案的纪实分析，一边是意象与呼告的诗化提炼——共同描绘出迁徙对一代人精神世界的双重重塑：既剥离了原有的归属，也催生了新的自我。年轻人正是在这种身心错位中艰难找到自己的「夹缝光芒」。

5评分要素 checklist

  • ✅ 4 段 结构（不是 3 段也不是 5 段）
  • ✅ 一段一文本（Body 1 = Text 1，Body 2 = Text 2，不混淆）
  • ✅ 每 Body 含 3 mini TEEL 循环（首先 / 其次 / 最后）
  • ✅ Conclusion 以 「综上所述」 开头
  • ✅ 文体匹配（非虚构 = 非虚构词汇；电影 = 电影词汇；不混用）
  • ✅ 总字数 ~800 字（25-mark），不少于 700 字
  • ❌ 是否出现「模棱两可」（"一方面......另一方面"——25 分必须一边倒）
  • ❌ 是否使用 个人化论据（"我的朋友......" "我妈妈说......"——使用权威证据）
  • ❌ 是否使用陈词滥调（马云 / 乔布斯 / 比尔盖茨——已被 HSC markers 列为论据陈旧）
  • ❌ 是否使用 hedge 词（可能 / 也许 / 或许）
⛔
禁词 / 禁套路清单
这些写法在 NESA HSC C&L 评分中被认定为自动降band
禁词类（任何题型）：
  • 「可能」「也许」「或许」 — hedge 词，降出 perceptive band
  • 用「体现 / 揭示 / 反讽地 / 折射出」替代
电影 vs 非虚构 vs 小说语体混用：
  • 分析夹缝中的光芒（非虚构）→ 禁用 镜头 / 场景调度 / 蒙太奇 / 剧本 / 台词 / 对白
  • 分析别告诉她（电影）→ 禁用 剧本 / 台词 / 对白 / script，必须用 镜头 / 画面 / 场景调度
论据类（Q4 重点）：
  • 个人化论据："我的朋友" / "我叔叔" / "我妈妈" → 必须用权威 / prescribed text 内的证据
  • 陈词滥调：马云 / 乔布斯 / 比尔盖茨 / 雷锋 → 已被 HSC markers 明确标记「论据陈旧」
结构类：
  • 模棱两可：「一方面......另一方面......」「不是简单的是或否」→ 25 分 Q4 必须立场鲜明
  • 首段过长：Intro > 80 字 → 论证开始太晚
  • 两张皮：技巧名 + 引文孤立罗列，无解释 → 不构成分析
  • 复述剧情：复述情节而非分析 → 不是「论证」而是「概述」
  • TEEL 不完整：缺 L 或缺 E → 每 cycle 都必须完整
