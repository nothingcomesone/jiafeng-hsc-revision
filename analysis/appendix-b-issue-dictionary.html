3Q3 题型方向

  • 作者如何呈现两代大学生在就业模式上的差异？
  • 作者如何通过具体家庭案例呈现城镇化进程的复杂性？
  • 作者如何通过对话引用和对比手法呈现不同地区年轻人职业观念的差异？

4Q4 essay 钩子

  • 「年轻人面对社会变革的回应方式决定了他们的成长轨迹」——通过夹缝中的光芒 vs 漂向北方/大江大河
  • 「个人在社会变革中寻求发展，需要抓住机会」——夹缝海燕/梅怡 + 大江大河雷东宝/杨巡
  • 「年轻人在制度更迭中如何重新定义安身立命的标准」——夹缝丽颖/素婷 + 二马马威
议题边界（不要写偏）：1.1 不包含「家庭代际矛盾」（属 1.3）/「女性独立」（属 1.2）/「教育扩张」（属 2.2）。学生易把 1.1 写成万能筐——必须聚焦「社会变化 → 个人冲击」这一因果方向。
学生陷阱：易把 1.1 写成空泛的「时代背景影响人」——必须具体到哪种社会变化（制度更迭 / 城镇化 / 市场化）和哪种个人冲击（身份、命运、价值观）。

  • 作者如何运用对比技巧呈现两代女性在性别角色认知上的转变？
  • 作者如何通过直接引语展现素婷的女性主义立场？
  • 作者如何通过陈倩的婚后选择探讨女性在家族企业中的新角色？

4Q4 essay 钩子

  • 「现代女性的成功标准已从外部确认转向内部自决」——夹缝素婷/陈倩/丽颖母 + 二马温都太太/伊牧师女儿
  • 「教育是女性突破传统命运链的核心杠杆」——夹缝海燕/梦兰 + 大江大河宋运萍
议题边界（不要写偏）：1.2 不是「女性故事」的统称——必须聚焦在「性别角色的变化」这一变量上。素婷的「拒绝婚房」是 1.2，但素婷的「跳槽哲学」是 2.1，不要混淆。
学生陷阱：易将所有女性人物笼统归为 1.2——其实每个女性人物在不同议题上的承载强度不同（见附录 C 矩阵）。

  • 「家庭分工从性别约定转向能力分工」——夹缝陈倩/丽颖母 + 二马温都母女
议题边界（不要写偏）：1.3 不等同于「家庭故事」——必须聚焦「家庭结构 / 分工 / 关系的变化」。家庭内部情感冲突单独存在不构成 1.3——必须连接到结构性转变。
学生陷阱：易将 1.3 与 1.2 混淆——丽颖母亲的觉醒同时涉及 1.2（女性独立）和 1.3（家庭分工重写）。学生应按题干引导明确锚定其中之一。

第 5 章丽颖「「劳务派遣」+「领导并不尊重」+「抑郁的阴霾」+「专门被坑的八零后」」制度性 + 心理双重压力

3Q3 题型方向

  • 作者如何通过对比与心理描写呈现丽颖职业发展中的困境与挑战？
  • 作者如何通过素婷的人生选择呈现年轻人应对压力的多元策略？
  • 作者如何通过具体数据细节突出梅怡创业选择的代价？

4Q4 essay 钩子

  • 「八零后并非整齐被动——他们在制度压力下分化出多种应对策略」——夹缝丽颖/素婷/梅怡 + 漂向北方
  • 「年轻人的压力源已从单一就业扩展到身份、关系、价值观多个维度」——夹缝 + 别告诉她碧莉的中西夹缝感
议题边界（不要写偏）：2.1 不限于「就业压力」——还包括身份不稳、人际焦虑、抑郁、孤独、代际责任等多维度压力。但不要把所有人物的负面情绪都笼统归为 2.1——必须连接到「当代社会」这一时代特征。
学生陷阱：易把 2.1 写成「丽颖很惨」「梅怡很苦」的故事汇报——必须分析压力的结构性来源（如劳务派遣制度、市场化、城乡差距）。

3Q3 题型方向

  • 作者如何通过素婷的人生哲学呈现后物质价值观在新一代中的兴起？
  • 作者如何呈现「张扬个性」「追求精神生活」等价值观在八零后中的扩散？

4Q4 essay 钩子

  • 「地域文化塑造代际价值观」——夹缝潮汕vs内地 + 漂向北方北漂vs故乡
  • 「八零后的价值观从物质成功转向精神生活」——夹缝素婷/梅怡 + 大江大河
议题边界（不要写偏）：2.3 不等同于「传统文化」——更聚焦「文化对年轻人的影响」，包括经济文化、消费文化、地域文化、代际文化。
学生陷阱：易将 2.3 写成「文化很重要」——必须指出哪一类文化（地域 / 经济 / 代际）+ 如何影响（价值观 / 行为 / 选择）+ 对哪一代（八零后 vs 七零后）。
